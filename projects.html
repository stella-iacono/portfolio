My Projects Interesting Websites About Me

Websites I Created:

  1. Hypertext Story

    2. In this website, I used "hypertext" to create a blog about a trip I went on. Using only HTML, I tell a little about places I visited.

  2. Internet Broadsheet

    3. This is a newspaper style broadsheet utilizing typography and grids. Each card contains a different pastry to bake and a link to the recipe.

  3. Transformation

    4. The goal of this website was to create a transformation using JavaScript. Select different features of cars and see an AI generated image of what it might look like. The background transforms when you toggle between 'retro' and 'modern'.

  4. Panting P5.js

    5. I recreated a painting by Paul Klee using a JavaScript library called p5.js. There are three versions of the painting - a static one, an animated one, and an interactive one.

  5. Campus Bingo

    6. This is a responsive website made for a mobile phone. It's specific to the WWU campus. Try to get five in a row!

  6. Top 20 Albums

    7. I created this webpage with a JSON file. All the details about my top 20 albums are displayed using JSON and JavaScript.

  7. Zee (elizabot.js)

    8. This site is a spin on the Eliza chatbot created by Joseph Weizenbaum in 1966. This version I created, Zee, needs the user to be its therapist.

  8. Zee Chatbot

    9. This was my final project for my Dynamic Webpages class. It is a re-imagined, updated version of my troubled chatbot, Zee. This version implements an API that I gave a prompt to act as a character I created, and also includes JavaScript.
